Scripts, soluciones y consejos

El sitio web complementario de este libro es http://www.nostarch.com/doingmathwithpython/. Aquí puedes descargar todos los programas de este libro, así como pistas y soluciones para los retos. También encontrarás enlaces a recursos adicionales de matemáticas, ciencias y Python que considero útiles, así como correcciones o actualizaciones del propio libro.

El software siempre está cambiando; una nueva versión de Python, SymPy o matplotlib puede hacer que una determinada funcionalidad demostrada en este libro se comporte de forma diferente. Cualquier cambio de este tipo se indicará en el sitio web.

Espero que este libro haga que tu viaje por la programación informática sea más divertido e inmediatamente relevante. ¡Hagamos matemáticas!
