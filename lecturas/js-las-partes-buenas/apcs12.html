Declaración var

JavaScript permite que las definiciones de var aparezcan en cualquier lugar dentro de una función. JSLint es más estricto.

JSLint espera que

  • Un var se declare sólo una vez, y que se declare antes de ser utilizado.

  • Una función se declare antes de ser utilizada.

  • Los parámetros no se declaren también como vars.

JSLint no espera

  • Que una matriz arguments se declare como var.

  • Que se declare una variable en un bloque. Esto se debe a que los bloques de JavaScript no tienen ámbito de bloque. Esto puede tener consecuencias inesperadas, así que define todas las variables al principio del cuerpo de la función.
