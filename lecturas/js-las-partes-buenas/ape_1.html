JavaScript Object Notation (JSON) es un formato ligero de intercambio de datos. Se basa en la notación literal de objetos de JavaScript, una de las mejores partes de JavaScript. Aunque es un subconjunto de JavaScript, es independiente del lenguaje. Puede utilizarse para intercambiar datos entre programas escritos en todos los lenguajes de programación modernos. Es un formato de texto, por lo que es legible por humanos y máquinas. Es fácil de implementar y de utilizar. Hay mucho material sobre JSON en http://www.JSON.org/.
