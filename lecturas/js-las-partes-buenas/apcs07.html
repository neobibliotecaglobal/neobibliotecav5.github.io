Bloques obligatorios

JSLint espera que las declaraciones if y for se hagan con bloques, es decir, con declaraciones encerradas entre llaves ({}).

JavaScript permite que un if se escriba así:

if (condition)
    statement;

Se sabe que esa forma contribuye a cometer errores en proyectos en los que muchos programadores trabajan en el mismo código. Por eso JSLint espera que se utilice un bloque:

if (condition) {
    statements;
}

La experiencia demuestra que esta forma es más resistente.
