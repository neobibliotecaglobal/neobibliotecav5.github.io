Actualizar

Un valor de un objeto puede actualizarse mediante una asignación. Si el nombre de la propiedad ya existe en el objeto, el valor de la propiedad se sustituye:

stooge['first-name'] = 'Jerome';

Si el objeto no tiene ya ese nombre de propiedad, el objeto se aumenta:

stooge['middle-name'] = 'Lester';
stooge.nickname = 'Curly';
flight.equipment = {
    model: 'Boeing 777'
};
flight.status = 'overdue';
