Eliminar

El operador delete se puede utilizar para eliminar una propiedad de un objeto. Eliminará una propiedad del objeto si la tiene. No tocará ninguno de los objetos del prototipo.

Eliminar una propiedad de un objeto puede permitir que aparezca una propiedad del prototipo de enlace:

another_stooge.nickname    // 'Moe'

// Remove nickname from another_stooge, revealing
// the nickname of the prototype.

delete another_stooge.nickname;

another_stooge.nickname    // 'Curly'
