hasPropiedadPropia

En el Capítulo 3, se ofreció el método hasOwnProperty como filtro para solucionar un problema con la declaración for in. Por desgracia, hasOwnProperty es un método, no un operador, por lo que en cualquier objeto podría sustituirse por una función diferente o incluso por un valor que no sea una función:

var name;
another_stooge.hasOwnProperty = null;       // trouble
for (name in another_stooge) {
    if (another_stooge.hasOwnProperty(name)) { // boom
        document.writeln(name + ': ' + another_stooge[name]);
    }
}
