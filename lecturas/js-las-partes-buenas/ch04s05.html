Retorno

Cuando se invoca una función, comienza su ejecución con la primera sentencia, y finaliza cuando llega a la } que cierra el cuerpo de la función. Eso hace que la función devuelva el control a la parte del programa que invocó la función.

La sentencia return puede utilizarse para hacer que la función regrese antes de tiempo. Cuando se ejecuta return, la función retorna inmediatamente sin ejecutar las sentencias restantes.

Una función siempre devuelve un valor. Si no se especifica el valor return, se devuelve undefined.

Si la función se invocó con el prefijo new y el valor return no es un objeto, entonces se devuelve this (el nuevo objeto).
