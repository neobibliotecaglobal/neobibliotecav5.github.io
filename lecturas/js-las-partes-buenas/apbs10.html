Envoltorios tipificados

JavaScript dispone de un conjunto de envolturas tipadas. Por ejemplo

new Boolean(false)

produce un objeto que tiene un método valueOf que devuelve el valor envuelto. Esto resulta completamente innecesario y, en ocasiones, confuso. No utilices new Boolean ni new Number ni new String.

Evita también new Object y new Array. Utiliza en su lugar {}y [].
