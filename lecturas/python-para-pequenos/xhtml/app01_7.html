La palabra clave break se utiliza para detener la ejecución de algún código. Podrías utilizar break dentro de un bucle for , así: 

age = 10
for x in range(1, 100):
    print(f'counting {x}')
    if x == age:
        print('end counting')
        break

Como aquí la variable age tiene el valor 10, este código imprimirá lo siguiente: 

counting 1
counting 2
counting 3
counting 4
counting 5
counting 6
counting 7
counting 8
counting 9
counting 10
end counting

Una vez que el valor de la variable x alcance 10, el código imprimirá el texto "fin del recuento" y saldrá del bucle.
