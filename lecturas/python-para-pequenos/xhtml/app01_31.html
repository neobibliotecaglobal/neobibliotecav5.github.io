Mientras que

La palabra clave while es un poco como for , salvo que un bucle for cuenta a través de un rango (de números), pero un bucle while sigue ejecutándose mientras una expresión es verdadera. Ten cuidado con los bucles while : si la expresión siempre es verdadera, el bucle nunca terminará (esto se llama bucle infinito ). He aquí un ejemplo: 

>>> x = 1
>>> while x == 1:
        print('hello')

Si ejecutas este código, formará un bucle eterno, o al menos hasta que cierres la Shell de Python o pulses CTRL-C para interrumpirlo. Sin embargo, el código siguiente imprimirá "hola" nueve veces (cada vez sumando 1 a la variable x , hasta que x ya no sea menor que 10): 

>>> x = 1
>>> while x < 10:
        print('hello')
        x = x + 1
