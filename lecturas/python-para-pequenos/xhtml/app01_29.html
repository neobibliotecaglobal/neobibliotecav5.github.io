Devolver

La palabra clave return se utiliza para devolver un valor de una función. Por ejemplo, podrías crear una función para calcular el número de segundos que has estado vivo hasta tu último cumpleaños: 

def age_in_seconds(age_in_years):
    return age_in_years * 365 * 24 * 60 * 60

Cuando llames a esta función, el valor devuelto se puede asignar a otra variable o se puede imprimir: 

>>> seconds = age_in_seconds(9)
>>> print(seconds)
283824000
>>> print(age_in_seconds(12))
378432000
