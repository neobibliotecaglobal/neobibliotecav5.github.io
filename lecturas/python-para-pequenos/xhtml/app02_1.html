Funciones incorporadas de Python

Python tiene una caja bien surtida de herramientas de programación, incluyendo un gran número de funciones y módulos que están listos para que los utilices. Estas herramientas incorporadas pueden hacer que escribir programas sea mucho más fácil.

Como leíste en el Capítulo 7 , los módulos deben importarse antes de poder utilizarse. Las funciones incorporadas de Python no necesitan importarse primero; están disponibles en cuanto se inicia el Shell de Python. En este apéndice, veremos algunas de las funciones incorporadas más útiles, y luego nos centraremos en una: la función open , que te permite abrir archivos para leer y escribir en ellos.
