La palabra clave and se utiliza para unir dos expresiones en una sentencia (como una sentencia if ) para decir que ambas expresiones deben ser verdaderas. Aquí tienes un ejemplo: 

if age > 12 and age < 20:
    print('Beware the teenager!!!!')

Este código significa que el valor de la variable age debe ser mayor que 12 y menor que 20 para que se imprima el mensaje.
