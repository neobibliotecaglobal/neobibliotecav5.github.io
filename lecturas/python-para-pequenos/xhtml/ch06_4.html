Lo que has aprendido

En este capítulo, hemos utilizado dos tipos de bucles para realizar tareas repetitivas: los bucles for y los bucles while . Son similares, pero pueden utilizarse de formas distintas. Le dijimos a Python lo que queríamos repetir escribiendo las tareas dentro de bloques de código, que pusimos dentro de bucles. También utilizamos la palabra clave break para detener el bucle.
