Es

La palabra clave is es un poco como el operador igual a ( == ), que se utiliza para saber si dos cosas son iguales (por ejemplo, 10 == 10 es verdadero y 10 == 11 es falso). Sin embargo, hay una diferencia fundamental entre is y == . Si estás comparando dos objetos (como listas), == puede devolver true , mientras que is puede que no (aunque pienses que los objetos son iguales). Se trata de un concepto avanzado de programación.
