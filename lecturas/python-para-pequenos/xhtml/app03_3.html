Error de atributo con Turtle

Algunos programadores noveles experimentan extraños errores de atributo al intentar utilizar tortuga: 

>>> import turtle
>>> t = turtle.Turtle()
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: module 'turtle' has no attribute 'Turtle'

La causa más típica de este error es que hayas creado un archivo llamado tortuga .py en tu carpeta de inicio. En este caso, cuando introduces import turtle , obtienes el archivo que has creado y no el módulo turtle de Python. Si eliminas o cambias el nombre de ese archivo, el módulo correcto debería importarse correctamente.
