Assert

La palabra clave assert se utiliza para decir que un valor debe ser verdadero. Es otra forma de detectar errores y problemas en el código, normalmente en programas más avanzados (por eso no utilizamos assert en Python para niños ). Aquí tienes una simple afirmación assert : 

>>> mynumber = 10
>>> assert mynumber < 5
Traceback (most recent call last):
  File "<pyshell#1>", line 1, in <module>
    assert mynumber < 5
AssertionError

En este ejemplo, afirmamos que el valor de la variable mynumber es menor que 5. No lo es, por lo que Python muestra un error (llamado AssertionError ).
