Lo que has aprendido

En este capítulo has aprendido cómo Python utiliza cadenas para almacenar texto y cómo utiliza listas y tuplas para manejar múltiples elementos. Has visto que los elementos de las listas pueden modificarse, y que puedes unir una lista a otra lista, pero que los valores de una tupla no pueden cambiar. También aprendiste a utilizar diccionarios para almacenar valores con claves que los identifican.
