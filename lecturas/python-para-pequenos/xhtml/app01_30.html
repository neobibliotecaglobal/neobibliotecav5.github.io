La palabra clave try inicia un bloque de código que termina con las palabras clave except y finally . Juntos, estos bloques de código try / except / finally se utilizan para gestionar errores en un programa, como por ejemplo asegurarse de que el programa muestra un mensaje útil al usuario en lugar de un error poco amigable de Python. Son muy útiles en programas avanzados.
