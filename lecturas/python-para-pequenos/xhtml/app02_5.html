La función any

La función any es similar a all , salvo que si alguno de los valores se evalúa como True , devolverá True . Intentemos el mismo ejemplo con los números: 

>>> mylist = [1, 2, 5, 6]
>>> any(mylist)
True

Nuestra lista mixta de ceros, None , cadenas vacías y False también funciona igual que all : 

>>> mylist = [0, False, None, "", 0, False, '']
>>> any(mylist)
False

Pero si hacemos un pequeño cambio en esa lista -como añadir un número distinto de cero, como 100- obtendremos True : 

>>> mylist = [0, False, None, "", 0, False, '', 100]
>>> any(mylist)
True
