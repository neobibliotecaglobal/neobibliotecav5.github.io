La palabra clave in se utiliza en expresiones para ver si un elemento está dentro de una colección de elementos. Por ejemplo, ¿se puede encontrar el número 1 en una lista (una colección) de números? 

>>> if 1 in [1,2,3,4]:
        print('number is in the list')
number is in the list

He aquí cómo averiguar si la cadena ’pants’ está en una lista de prendas de vestir: 

>>> clothing_list = ['shorts', 'undies', 'boxers', 'long johns',
                 'knickers']
>>> if 'pants' in clothing_list:
        print('pants is in the list')
    else:
        print('where are my pants?')
where are my pants?
