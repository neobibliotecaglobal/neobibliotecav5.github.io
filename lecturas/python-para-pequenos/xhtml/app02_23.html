La función suma

La función sum suma los elementos de una lista y devuelve el total. Aquí tienes un ejemplo: 

>>> my_list_of_numbers = list(range(0, 500, 50))
>>> print(my_list_of_numbers)
[0, 50, 100, 150, 200, 250, 300, 350, 400, 450]
>>> print(sum(my_list_of_numbers))
2250

En la primera línea, creamos una lista de números entre 0 y 500, utilizando range con un paso de 50. A continuación, imprimimos la lista para ver el resultado. Por último, pasando la variable my_list_of_numbers a la función sum con print(sum(my_list_of_numbers)) sumamos todos los elementos de la lista, obteniendo el total de 2250.
