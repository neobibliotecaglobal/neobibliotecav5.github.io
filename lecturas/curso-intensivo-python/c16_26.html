Otras escalas de color

Puedes elegir entre otras escalas de color. Para ver las escalas de color disponibles, introduce las dos líneas siguientes en una sesión de terminal Python:

>>> import plotly.express as px
>>> px.colors.named_colorscales()
['aggrnyl', 'agsunset', 'blackbody', ..., 'mygbm']

No dudes en probar estas escalas de color en el mapa de terremotos, o con cualquier conjunto de datos en el que la variación continua de los colores pueda ayudar a mostrar patrones en los datos.
