Cómo dar estilo a tus sentencias if

En todos los ejemplos de este capítulo has visto buenos hábitos de estilo. La única recomendación que ofrece PEP 8 para dar estilo a las pruebas condicionales es utilizar un solo espacio alrededor de los operadores de comparación, como ==, >= y <=. Por ejemplo

if age < 4:

es mejor que:

if age<4:

Este espaciado no afecta a la forma en que Python interpreta tu código; sólo hace que tu código sea más fácil de leer para ti y para los demás.
