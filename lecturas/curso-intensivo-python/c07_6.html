El operador módulo

Una herramienta útil para trabajar con información numérica es el modulo operator (%), que divide un número por otro y devuelve el resto:

>>> 4 % 3
1
>>> 5 % 3
2
>>> 6 % 3
0
>>> 7 % 3
1

El operador módulo no te dice cuántas veces cabe un número en otro; sólo te dice cuál es el resto.

Cuando un número es divisible por otro, el resto es 0, por lo que el operador módulo siempre devuelve 0. Puedes utilizar este hecho para determinar si un número es par o impar:

even_or_odd.py

number = input("Enter a number, and I'll tell you if it's even or odd: ")
number = int(number)

if number % 2 == 0:
    print(f"\nThe number {number} is even.")
else:
    print(f"\nThe number {number} is odd.")

Los números pares siempre son divisibles por dos, por lo que si el módulo de un número y dos es cero (aquí, if number % 2 == 0) el número es par. En caso contrario, es impar.

Enter a number, and I'll tell you if it's even or odd: 42

The number 42 is even.
