Versiones de Python

Todo lenguaje de programación evoluciona a medida que surgen nuevas ideas y tecnologías, y los desarrolladores de Python han hecho que el lenguaje sea cada vez más versátil y potente. En el momento de escribir estas líneas, la última versión es Python 3.11, pero todo lo que aparece en este libro debería funcionar con Python 3.9 o posterior. En esta sección, averiguaremos si Python ya está instalado en tu sistema y si necesitas instalar una versión más reciente. El Apéndice A también contiene detalles adicionales sobre la instalación de la última versión de Python en cada uno de los principales sistemas operativos.
