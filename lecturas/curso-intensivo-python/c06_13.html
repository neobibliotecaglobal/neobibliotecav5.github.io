Nota

Si omites el segundo argumento en la llamada a get() y la clave no existe, Python devolverá el valor None. El valor especial None significa "no existe ningún valor". No es un error: es un valor especial para indicar la ausencia de un valor. Verás más usos de None en el Capítulo 8.
