Guiones bajos en los números

Cuando escribas números largos, puedes agrupar los dígitos utilizando guiones bajos para que los números grandes sean más legibles:

>>> universe_age = 14_000_000_000

Cuando imprimes un número definido con guiones bajos, Python sólo imprime los dígitos:

>>> print(universe_age)
14000000000

Python ignora los guiones bajos cuando almacena este tipo de valores. Aunque no agrupes los dígitos de tres en tres, el valor no se verá afectado. Para Python, 1000 es lo mismo que 1_000, que es lo mismo que 10_00. Esta característica funciona tanto para enteros como para flotantes.
