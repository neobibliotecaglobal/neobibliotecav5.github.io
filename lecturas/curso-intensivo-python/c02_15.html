Eliminar prefijos

Al trabajar con cadenas, otra tarea común es eliminar un prefijo. Considera una URL con el prefijo común https://. Queremos eliminar este prefijo para centrarnos sólo en la parte de la URL que los usuarios deben introducir en la barra de direcciones. He aquí cómo hacerlo:

>>> nostarch_url = 'https://nostarch.com'
>>> nostarch_url.removeprefix('https://')
'nostarch.com'

Introduce el nombre de la variable seguido de un punto y, a continuación, el método removeprefix(). Dentro del paréntesis, introduce el prefijo que quieras eliminar de la cadena original.

Al igual que los métodos para eliminar espacios en blanco, removeprefix() no modifica la cadena original. Si quieres conservar el nuevo valor con el prefijo eliminado, reasígnalo a la variable original o asígnalo a una nueva variable:

>>> simple_url = nostarch_url.removeprefix('https://')

Cuando veas una URL en una barra de direcciones y no aparezca la parte https://, es probable que el navegador esté utilizando un método como removeprefix() entre bastidores.
