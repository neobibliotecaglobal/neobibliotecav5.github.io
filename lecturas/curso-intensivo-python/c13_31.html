Identificar cuándo deben ejecutarse las partes del juego

Tenemos que identificar las partes del juego que deben ejecutarse siempre y las partes que deben ejecutarse sólo cuando el juego está activo:

alien_invasion.py

    def run_game(self):
        """Start the main loop for the game."""
        while True:
            self._check_events()

            if self.game_active:
                self.ship.update()
                self._update_bullets()
                self._update_aliens()

            self._update_screen()
            self.clock.tick(60)

En el bucle principal, siempre tenemos que llamar a _check_events(), aunque el juego esté inactivo. Por ejemplo, seguimos necesitando saber si el usuario pulsa Q para salir del juego o si hace clic en el botón para cerrar la ventana. También seguimos actualizando la pantalla para poder hacer cambios en ella mientras esperamos a ver si el jugador decide empezar una nueva partida. El resto de las llamadas a funciones sólo tienen que ocurrir cuando el juego está activo, porque cuando el juego está inactivo, no necesitamos actualizar las posiciones de los elementos del juego.

Ahora, cuando juegues a Alien Invasion, el juego debería congelarse cuando hayas utilizado todas tus naves.
