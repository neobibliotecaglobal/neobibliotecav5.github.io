Instalar Plotly

Instala Plotly con pip, igual que hiciste con Matplotlib:

$ python -m pip install --user plotly
$ python -m pip install --user pandas

Plotly Express depende de pandas, que es una biblioteca para trabajar eficientemente con datos, así que también tenemos que instalarla. Si utilizaste python3 u otra cosa al instalar Matplotlib, asegúrate de utilizar el mismo comando aquí.

Para ver qué tipo de visualizaciones son posibles con Plotly, visita la galería de tipos de gráficos en https://plotly.com/python. Cada ejemplo incluye el código fuente, para que puedas ver cómo Plotly genera las visualizaciones.
