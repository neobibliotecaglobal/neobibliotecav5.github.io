Comprobación de la desigualdad

Cuando quieras determinar si dos valores no son iguales, puedes utilizar la sentencia inequality operator (!=). Utilicemos otra sentencia if para examinar cómo utilizar el operador de desigualdad. Vamos a almacenar un ingrediente de pizza solicitado en una variable y luego imprimiremos un mensaje si la persona no pidió anchoas:

toppings.py

requested_topping = 'mushrooms'

if requested_topping != 'anchovies':
    print("Hold the anchovies!")

Este código compara el valor de requested_topping con el valor 'anchovies'. Si estos dos valores no coinciden, Python devuelve True y ejecuta el código que sigue a la sentencia if. Si los dos valores coinciden, Python devuelve False y no ejecuta el código que sigue a la sentencia if.

Como el valor de requested_topping no es 'anchovies', se ejecuta la función print():

Hold the anchovies!

La mayoría de las expresiones condicionales que escribas comprobarán la igualdad, pero a veces te resultará más eficaz comprobar la desigualdad.
