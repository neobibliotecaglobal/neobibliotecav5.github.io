Comprobación del registro

Git mantiene un registro de todas las confirmaciones realizadas en el proyecto. Vamos a comprobar el registro:

git_practice$ git log
commit cea13ddc51b885d05a410201a54faf20e0d2e246 (HEAD -> main)
Author: eric <eric@example.com>
Date:   Mon Jun 6 19:37:26 2022 -0800

    Started project.
git_practice$

Cada vez que realizas una confirmación, Git genera un ID de referencia único de 40 caracteres. Registra quién hizo la confirmación, cuándo se hizo y el mensaje registrado. No siempre necesitarás toda esta información, por lo que Git proporciona una opción para imprimir una versión más simple de las entradas del registro:

git_practice$ git log --pretty=oneline
cea13ddc51b885d05a410201a54faf20e0d2e246 (HEAD -> main) Started project.
git_practice$

La bandera --pretty=oneline proporciona las dos piezas de información más importantes: el ID de referencia de la confirmación y el mensaje registrado para la confirmación.
