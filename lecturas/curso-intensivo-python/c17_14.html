Añadir enlaces clicables

Como Plotly te permite utilizar HTML en elementos de texto, podemos añadir fácilmente enlaces a un gráfico. Utilicemos las etiquetas del eje x para que el espectador pueda visitar la página de inicio de cualquier proyecto en GitHub. Tenemos que extraer las URL de los datos y utilizarlas al generar las etiquetas del eje x:
