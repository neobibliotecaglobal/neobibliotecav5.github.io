Actualizar pip

Python incluye una herramienta llamada pip que se utiliza para instalar paquetes de terceros. Como pip ayuda a instalar paquetes de recursos externos, se actualiza a menudo para solucionar posibles problemas de seguridad. Así que empezaremos por actualizar pip.

Abre una nueva ventana de terminal y ejecuta el siguiente comando:

$ python -m pip install --upgrade pip
❶ Requirement already satisfied: pip in /.../python3.11/site-packages (22.0.4)
--snip--
❷ Successfully installed pip-22.1.2

La primera parte de este comando, python -m pip, indica a Python que ejecute el módulo pip. La segunda parte, install --upgrade, indica a pip que actualice un paquete ya instalado. La última parte, pip, especifica qué paquete de terceros debe actualizarse. La salida muestra que mi versión actual de pip, la 22.0.4 ❶, fue sustituida por la última versión en el momento de escribir esto, la 22.1.2 ❷.

Puedes utilizar este comando para actualizar cualquier paquete de terceros instalado en tu sistema:

$ python -m pip install --upgrade package_name
