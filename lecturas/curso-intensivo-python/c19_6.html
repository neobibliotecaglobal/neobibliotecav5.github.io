19-1. Blog: Inicia un nuevo proyecto Django llamado Blog. Crea una aplicación llamada blogs, con un modelo que represente un blog en general, y un modelo que represente una entrada individual del blog. Asigna a cada modelo un conjunto adecuado de campos. Crea un superusuario para el proyecto y utiliza el sitio de administración para crear un blog y un par de entradas cortas. Crea una página de inicio que muestre todas las entradas en el orden adecuado.

Crea páginas para crear un blog, para crear nuevas entradas y para editar las entradas existentes. Utiliza tus páginas para asegurarte de que funcionan.
