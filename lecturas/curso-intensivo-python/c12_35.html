Añadir la configuración de las balas

Al final del método __init__(), actualizaremos settings.py para incluir los valores que necesitaremos para una nueva clase Bullet:

settings.py

    def __init__(self):
        --snip--
        # Bullet settings
        self.bullet_speed = 2.0
        self.bullet_width = 3
        self.bullet_height = 15
        self.bullet_color = (60, 60, 60)

Estos valores crean balas de color gris oscuro con una anchura de 3 píxeles y una altura de 15 píxeles. Las balas se desplazarán ligeramente más rápido que la nave.
