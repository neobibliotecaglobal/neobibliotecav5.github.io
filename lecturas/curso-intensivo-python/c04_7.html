Olvidar la sangría

Sangrar siempre la línea que sigue a la sentencia for en un bucle. Si te olvidas, Python te lo recordará:

magicians.py

magicians = ['alice', 'david', 'carolina']
for magician in magicians:
❶ print(magician)

La llamada a print() ❶ debería estar sangrada, pero no lo está. Cuando Python espera un bloque indentado y no lo encuentra, te indica con qué línea ha tenido un problema:

  File "magicians.py", line 3
    print(magician)
    ^
IndentationError: expected an indented block after 'for' statement on line 2

Normalmente puedes resolver este tipo de error de indentación indentando la línea o líneas inmediatamente posteriores a la sentencia for.
