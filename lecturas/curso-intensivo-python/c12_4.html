Instalación de Pygame

Antes de empezar a programar, instala Pygame. Lo haremos del mismo modo que instalamos pytest en el Capítulo 11: con pip. Si te has saltado el Capítulo 11 o necesitas un repaso sobre pip, consulta "Instalar pytestcon pip" en la página 210.

Para instalar Pygame, introduce el siguiente comando en una terminal:

$ python -m pip install --user pygame

Si utilizas un comando distinto de python para ejecutar programas o iniciar una sesión de terminal, como python3, asegúrate de utilizar ese comando en su lugar.
