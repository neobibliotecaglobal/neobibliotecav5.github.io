¿Cómo se escriben los comentarios?

En Python, la almohadilla (#) indica un comentario. Cualquier cosa que siga a una almohadilla en tu código es ignorada por el intérprete de Python. Por ejemplo:

comment.py

# Say hello to everyone.
print("Hello Python people!")

Python ignora la primera línea y ejecuta la segunda.

Hello Python people!
