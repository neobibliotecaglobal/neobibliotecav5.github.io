Tirar el dado

Antes de crear una visualización basada en la clase Die, vamos a lanzar un D6, imprimir los resultados y comprobar que los resultados parecen razonables:

die_visual.py

from die import Die

# Create a D6.
❶ die = Die()

# Make some rolls, and store results in a list.
results = []
❷ for roll_num in range(100):
    result = die.roll()
    results.append(result)

print(results)

Creamos una instancia de Die con los seis lados por defecto ❶. A continuación, lanzamos el dado 100 veces ❷ y almacenamos el resultado de cada tirada en la lista results. Aquí tienes un ejemplo de resultados:

Un examen rápido de estos resultados muestra que la clase Die parece funcionar.
