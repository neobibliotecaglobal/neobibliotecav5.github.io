Creación de la clase RandomWalk

Para crear un paseo aleatorio, crearemos una clase RandomWalk, que tomará decisiones aleatorias sobre la dirección que debe tomar el paseo. La clase necesita tres atributos: una variable para registrar el número de puntos del paseo, y dos listas para almacenar las coordenadas x- y y- de cada punto del paseo.

Sólo necesitaremos dos métodos para la clase RandomWalk: el método __init__() y fill_walk(), que calculará los puntos del paseo. Empecemos por el método __init__():
